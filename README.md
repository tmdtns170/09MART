# 09MART  
지역 기반 공동구매 및 배송 플랫폼

## 📌 프로젝트 개요
09MART는 지역 내 소비자들이 공동구매를 통해 배송비 부담을 줄이고,  
판매자·구매자·배송기사가 효율적으로 협업할 수 있도록 설계된  
지역 기반 공동구매 플랫폼입니다.

---

## ⚙ 주요 기능

### 🏪 판매자
- 상품 등록 및 관리
- 판매 현황 조회 (조회수 / 주문량 차트)
- 구매자 댓글 알림 및 답글
- 판매자 정보 관리
  
### 🛒 구매자
- 회원가입 및 로그인
- 공동구매 상품 조회 및 구매
- 주문 내역 조회 (준비중 / 배송중 / 배송완료)
- 리뷰 작성 및 알림 수신
- 즐겨찾기 기능

### 🚚 배송기사
- 배송 목록 확인 및 배송 배정 정보 조회
- 배송 경로 및 지도 기반 최적 동선 확인
- 배송 상태 변경 (배송중 → 배송완료)
- 배송 완료 시 구매자 알림 전송
- 배송 내역 조회
- 관리자와 실시간 채팅 (Socket 기반)
  
### 🛠 관리자
- 공지사항 관리
- Q&A 관리
- 상품 관리
- 배송기사 채팅 관리 (Socket 기반)
- 월별 가입자 및 방문자 통계 관리

## 🗄 데이터베이스 설계 (ERD)





본 프로젝트는 사용자, 상품, 주문, 배송을 중심으로  
관계형 데이터베이스 구조를 설계하였으며,  
**공동구매 주문 흐름과 배송 관리 로직을 고려하여 총 17개의 테이블로 구성**하였습니다.

- 사용자 / 판매자 / 관리자 역할 분리
- 주문–배송 흐름 중심의 테이블 관계 설계
- 데이터 중복 최소화 및 확장성 고려
